Geffen Peer Tutoring Scheduler


What we can do:

  1. Create a tutoring schedule
  2. Request tutor and student changes
  3. Automatically send emails to students and tutors

Directions

  1. Go to both the Google Sheets with the student responses and tutor responses.
  2. Make sure both Google Sheet links are accessible to everyone.
  3. Copy link and go to Setup page.
  4. Go to the Setup and copy each respected link.
  5. After setup is complete, go to the Search tab to view student assignments and request changes.
  6. Go to the Email tab to view and automatically send emails to students and tutors.
