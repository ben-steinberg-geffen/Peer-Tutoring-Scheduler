Geffen Peer Tutoring Scheduler


What we can do:

  1. Create a tutoring schedule
  2. Request tutor and student changes
  3. Automatically send emails to students and tutors

Directions

  1. Go to the Google Sheets with the student responses
  2. Go to file -> download -> .csv
  3. Repeat this process for the tutor responses sheet
  4. Go to the Setup tab and upload both files
  5. After upload is complete, go to the Search tab to view student assignments and request changes
  6. Go to the Email tab to view and automatically send emails to students and tutors
